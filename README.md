## Notice: GitHub Closure & Platform Updates

Dear visitors and community members,  

We would like to inform you that Xyra Studios has officially closed its GitHub page. This means that we will no longer be maintaining projects, boards, or updates on GitHub.  

GitHub has been a valuable platform where we shared projects, updates, and tools with our community. However, we have decided to migrate all services to a dedicated, modern platform to improve user experience.  

Our new platform allows us to provide a more secure, streamlined, and interactive environment for all users.  

All ongoing projects, submissions, and reports will now be exclusively hosted on our new website.  

We encourage all visitors to bookmark and access our new website regularly to stay updated with the latest features and announcements.  

The new platform supports a modern interface, faster updates, and enhanced security for all users.  

Additionally, the new website provides features that were not possible on GitHub, including a dedicated admin panel and interactive public dashboards.  

For contributors who followed our GitHub repositories, please note that we will no longer accept contributions, forks, or issues here.  

This decision ensures that we can maintain a consistent experience across all services while focusing on the growth and functionality of our new platform.
